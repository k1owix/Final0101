# Читай город

Учебное решение для книжного магазина `Читай город` на `C# / .NET 8`.

В состав входят:

- `DataAccess` — модели и сервисы доступа к SQL Server
- `WebApi` — ASP.NET Core API для книг, пользователей и заказов
- `DesktopApp` — WPF-клиент для просмотра каталога и оформления заказов

## О проекте

Функциональность:

    Просмотр каталога товаров в виде карточек
    Фильтрация и сортировка товаров
    Авторизация пользователей (Администратор, Менеджер, Клиент)
    Формирование и оформление заказов
    Генерация талона с кодом получения
    Управление заказами для сотрудников
    Веб-приложение для просмотра каталога


## Подключение к базе

Строка подключения хранится в [appsettings.json]:

```json
"DefaultConnection": "Server=.\\SQLEXPRESS;Database=ispp4104;Trusted_Connection=True;Encrypt=False;TrustServerCertificate=True;"
```

`http://localhost:5022/swagger`

Для корректной работы клиенту нужен запущенный `WebApi`.
